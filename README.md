# Customer LTV vs Acquisition Cost — Seaborn Visualization

Contact: 24f2001869@ds.study.iitm.ac.in

This repository contains a professional Seaborn-based visualization for customer analytics.

Files:
- `README.md` — this readme file (contains contact email).
- `chart.py` — Python script that generates the Seaborn scatterplot.
- `chart.png` — Generated scatterplot image (exactly 512x512 pixels).

How to run:
1. Install dependencies:
   ```bash
   pip install seaborn matplotlib pandas numpy Pillow
   ```
2. Run the script:
   ```bash
   python chart.py
   ```
3. The script will generate `chart.png` in the same folder.
